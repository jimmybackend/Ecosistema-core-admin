RESPONSIVE DESIGN - BREAKPOINTS:
- Extra small (xs): < 576px (Mobile portrait)
- Small (sm): ≥ 576px (Mobile landscape)
- Medium (md): ≥ 768px (Tablet portrait)
- Large (lg): ≥ 992px (Tablet landscape/Desktop small)
- Extra large (xl): ≥ 1200px (Desktop)
- Extra extra large (xxl): ≥ 1400px (Large desktop)

COMPORTAMIENTOS RESPONSIVE:
- Mobile First approach
- Sidebar se convierte en offcanvas/drawer en mobile
- Header se simplifica en mobile
- Tables se vuelven scrollable horizontal en mobile
- Cards stack en columnas en mobile
- Navigation se convierte en hamburger menu
- Forms se adaptan a pantallas pequeñas
- Images son responsive por defecto
- Touch targets mínimos de 44x44px
- Gestures support (swipe, tap, long press)

NOTAS ADICIONALES:
- Todos los colores deben estar en variables CSS
- Todos los tamaños deben usar rem o em units
- Transiciones de 0.3s con ease-in-out
- Box-sizing: border-box global
- Reset CSS moderno (normalize.css o similar)
- Font smoothing optimizado
- Subpixel rendering
- Text rendering optimizado
- Image rendering optimizado
- Backface visibility hidden para animaciones
- Will-change para elementos animados
- Hardware acceleration donde sea necesario
