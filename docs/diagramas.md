# Diagramas — referencia técnica mínima

Los diagramas narrativos/comerciales para presentación pública viven en:

- `jimmybackend/Ecosistema-presentacion`

En Core Admin se conserva la referencia técnica editable de operación real en:

- `assets/diagrams/`

Uso interno:

- diagramas para soporte técnico y onboarding operativo,
- sin datos reales ni secretos,
- alineados al estado real del sistema y no a promesas comerciales.

Ver política de canonicidad:

- `docs/project/PRESENTATION_REPOSITORY_POINTERS.md`
