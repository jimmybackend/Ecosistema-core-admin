# CORE Admin Route-Service-Repository-Table Map

Mapa técnico de mantenimiento para ubicar rápidamente, por ruta, el servicio/repositorio principal, vista asociada, tablas tocadas y estado operativo.

## Convenciones
- **Permiso requerido**: permiso explícito vía `requirePermission(...)` en la ruta; si no se observa validación explícita en esa ruta, se marca `autenticado (sin permiso explícito en esta ruta)` o `pública`.
- **CSRF requerido**: `sí` para `POST` con validación `ensureValidCsrfToken(...)`; `no` para rutas públicas sin token.
- **Tablas reales**: solo tablas confirmables desde SQL de repositorio o SQL inline en `routes/web.php`. Si no es trazable de forma directa aquí, se marca **no confirmado**.
- **Estado**: `read-only`, `dry-run`, `controlled`, `roadmap` (según comportamiento real observado).

## Clasificación rápida de rutas (seguridad operativa)

- **Públicas**: `/u/{slug}`, `/l/{slug}`, `POST /l/{slug}/forms/{id}/submit`, `POST /browser/analytics/collect` (técnica).
- **Autenticadas read-only**: vistas `GET` administrativas por módulo (`/dashboard`, `/tenants`, `/modules`, `/security/permissions-audit*`, etc.).
- **Autenticadas write**: rutas `POST` administrativas de CRUD/ejecución (`/users*`, `/roles*`, `/reports/exports`, `/workflow/*/execute`, etc.).
- **Dry-run GET/POST**: rutas explícitas `*dry-run*` de seguridad, workflow, reports, campaigns, CRM, landing, browser analytics y mail notifications.
- **Controlled write**: rutas de escritura real detrás de permiso + CSRF + flags (`/security/rate-limit/enforce`, `/workflow/*/execute`, `/campaigns`, `/reports/exports`, `/cloud/drive/upload`, etc.).

## Notas rápidas para mantenimiento
- Priorizar revisión de rutas `POST` con estado `controlled`: son los puntos de escritura real más sensibles.
- Las rutas públicas (`/u/*`, `/l/*`) deben revisarse siempre junto a sus flags para evitar aperturas no deseadas.
- Cuando una tabla está en `no confirmado`, ir al repositorio de módulo (`app/Core/<Modulo>/*Repository.php`) y verificar SQL puntual antes de cambios de esquema.
